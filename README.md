# heiwork
Plataforma y comunidad para freelancers en español
Muy pronto estaremos publicando toda la info relacionada al proyecto (Heiwork) ,cuando tengamos mas desarrollada la plataforma.
Heiwork fue creada como una idea de comunidad de trabajadores freelancers donde puedan interactuar y conseguir proyectos en los cuales poder trabajar.
La comunidad esta creciendo en este año 2021 y estamos recibiendo profesionales freelancers de todas partes del mundo, estaremos atentos a las nuevas novedades para poder informarlas en este repositorio.
